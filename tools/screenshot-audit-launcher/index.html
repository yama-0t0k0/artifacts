固定仕様（毎回必ず適用）

  • 既存スクリーンショットは問答無用で置き換え — 出力先フォルダが既にあれば確認なしで全削除してから撮り直す。履歴は残さず、same ブランチには常に最新版のみ。
  • 本番 Firebase への書き込み禁止 — 保存・送信・公開切替・応募・面談承諾/お断り・チャット送信・ログアウト等のボタンは絶対に押さない。
  • git はスクリーンショットフォルダのみ — ユーザーの未コミット変更（App.js 等）は絶対にステージしない。
